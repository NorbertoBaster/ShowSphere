Title: ShowSphere App

Description:
A movie and TV discovery app built with Next.js and TypeScript.
Users can filter by genre, sort, year, and rating. Filters are URL-synced for shareable links and refresh-safe state. Includes an accessible modal for detailed movie/TV info.

Key Features:

URL-driven filters: deep linking and refresh persistence

Accessible modals: keyboard navigation, focus trap, ARIA attributes

Skeleton loading & empty states: polished user experience

Responsive design & hover animations: modern, professional feel

Tech Stack:
Next.js, TypeScript, Tailwind (or inline CSS), TMDB API

Takeaways / Skills Demonstrated:

Full-stack thinking in a front-end app

Advanced state management & URL synchronization

Accessibility best practices

UX polish and performance awareness
